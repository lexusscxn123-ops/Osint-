🔎 Open Source Intelligence Tool

LEXUS OSINT TOOL; kullanıcı adı ve fotoğraflar üzerinden açık kaynaklı bilgileri analiz etmeye yardımcı olan basit bir araçtır.

---

⚡ Özellikler

- 🔎 Username OSINT (sosyal medya arama)
- 🖼 Fotoğraftan EXIF metadata analizi
- 🌐 Açık kaynak platform kontrolü
- 🧠 Basit terminal menüsü

---

📦 Kurulum

git clone https://github.com/lexusscxn123-ops/Osint-
cd Osint-
pip install -r requirements.txt

---

▶ Çalıştırma

python osint.py
